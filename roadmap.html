Glossary
Từ Điển
Thuật Ngữ Kỹ Thuật
ES6
Phiên bản JavaScript hiện đại với cú pháp mới: arrow function, destructuring, import/export, template literals.
DOM
Document Object Model — cấu trúc HTML mà JS dùng để đọc và thay đổi nội dung trang web.
API
Application Programming Interface — dịch vụ trả dữ liệu về cho ứng dụng qua các HTTP request.
CRUD
Create, Read, Update, Delete — bốn thao tác cơ bản khi làm việc với dữ liệu.
async/await
Cú pháp viết code bất đồng bộ trông như đồng bộ, dễ đọc hơn so với callback hay Promise thuần.
axios
Thư viện gửi HTTP request (GET/POST/PUT/DELETE) tới API, xử lý response gọn gàng hơn fetch.
debounce
Kỹ thuật trì hoãn hàm để tránh gọi quá nhiều lần liên tiếp — thường dùng cho ô tìm kiếm.
import/export
Cú pháp ES6 để chia code thành các module riêng biệt, dễ bảo trì và tái sử dụng.
Minimax
Thuật toán AI trong game: duyệt cây quyết định để chọn nước đi tối ưu bằng cách tối đa/tối thiểu điểm.
Alpha-Beta
Tối ưu hóa Minimax — cắt bỏ các nhánh không ảnh hưởng kết quả, giảm thời gian tính toán.
